### [简介](https://github.com/sochub/CH56/wiki) 

[CH561](https://github.com/sochub/CH561) 是一款32位RISC精简指令集CPU，默认系统主频为100MHZ, 64K FLASH、28K DATAFLASH、32K SRAM, 片上集成支持10/100M自适应的网络接口（MAC + PHY），内置网络PHY，LQFP64M，可广泛应用于各种联网控制类应用。
