### [描述](https://github.com/sochub/CH56/wiki) 

CH56x系列是WCH 32位高性能型高速接口系列MCU，该系列运行频率可以在120MHz左右，包括的主要特殊功能有：USB / Ethernet / SATA / SDIO

该系列的特点就是非常见的的数据通信接口，可扩展的存储介质接口，适合用于USB转存储设备。
