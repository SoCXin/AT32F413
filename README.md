### 关键特性

* Cortex®-M4主频最高 200 MHz
* 高达 16MB 外部 SPI Flash memory 扩展
* 3 x USART + 2 x UART
* 2 x CAN 2.0B
* Crystal-less USB2.0
* SDIO
